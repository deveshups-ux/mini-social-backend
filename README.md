# Mini Social Backend

A mini social media backend project built using Node.js, Express.js, MongoDB, and EJS.

## Features

- User Registration & Login
- Password hashing using bcrypt
- JWT Authentication
- Create Posts
- Edit Posts
- Like / Unlike Posts
- User Profile showing all posts
- MongoDB relationships using populate()

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- JWT
- bcrypt
